# Kata, Pluto Rover

### Dependencies
 - Go

### Installation
 - run `go get github.com/egnwd/kata`

### Building
 - run `go build ./...`

### Testing
 - run `go test ./...`

### Running
 - run `go install`
 - then `kata command`
 - `command` can be any string made up of any number of `F`s, `B`s, `L`s & `R`s
 - Example: `kata FFRFRFFLFLFF`
